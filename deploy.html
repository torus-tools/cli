Arjan Deploy

Arjan deploy is a tool that helps you deploy static sites with CDN and HTTPS to the AWS Cloud using a single command.

Get Started Docs

Leverage the power and cost savings of the AWS Cloud in your static sites

How it Works

Arjan deploy uses Cloudformation templates to help you provision and deploy all the necessary resources for your site in the AWS cloud. This means your entire stack and all of its resources are kept inside a JSON file which you can store locally and update continually.

Features

  • Customize the stack of your static website
  • Manage your project’s infrastructure with a single JSON file using AWS Cloudformation
  • Save changesets locally
  • Add and automatically verify SSL certificates with ACM
  • Import existing AWS Resources
  • Predefined templates for faster deployments

For an easier development workflow we've added some pre defined templates

Dev

S3 root bucket with a public policy.

Test

S3 root bucket, www reroute bucket and a route53 hosted zone.

Production

CDN w/ Route53 DNS (https): Deploys s3 bucket, route53 DNS, a cloudfront distribution and creates TLS certificates in AWS ACM.

Get Started

  1. If you haven't already, install Arjan globally
  2. Go into your project's directory
  3. Initialize the project (you only have to do this once per project)
  4. Deploy your project
