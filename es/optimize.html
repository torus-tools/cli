Arjan Optimize

Arjan optimize utiliza algunos de los mejores módulos node.js para compresión y minificación junto con sus propias características especiales.

Características

  • Comprime todo el activo estático con un solo comando
  • Elimina las clases CSS no utilizadas
  • Crear imágenes receptivas (para cada tipo de dispositivo)
  • Convertir imágenes a Webp con retrocesos para navegadores no compatibles
  • Soporta sintaxis ES6+
